# Лабораторная работа 5: Agile и Scrum

Учебный проект по созданию простого личного блога с использованием микрофrameworkа Flask. 
Цель работы — освоить практики Agile и Scrum через моделирование процесса в GitHub Issues. 
Блог позволит автору публиковать статьи, а посетителям — просматривать их. 
Процесс разработки имитируется одним спринтом с использованием всех основных артефактов Scrum.
